# Spinning Cube

> My blog post: https://www.spigis.com/blog/spinning-cube

This project was made with Go with the [tcell](https://github.com/gdamore/tcell) package.

You can check the math in [a1k0n's blog](https://www.a1k0n.net/2011/07/20/donut-math.html)

### How to run the project

Make sure you have [Go](https://go.dev/) installed

Clone the repo
```sh
git clone https://github.com/oliverTuesta/spinning-cube.git
```

Run the Square 2D
```sh
go run ./cmd/square2d
```
Run the Spinning Cube
```sh
go run ./cmd/cube3d
```
